One-time setup (host only)

  1. Create a new Google Sheet
  2. Extensions → Apps Script → delete default code → paste google-apps-script/Code.gs from this repo → Save
  3. In Apps Script, select setup → Run (authorize when asked)
  4. Deploy → New deployment → Web app
    Execute as: Me · Who has access: Anyone
  5. Copy the Web App URL → paste above → Connect & Sync
  6. Share the Sheet with your friends (view or edit). Share the bracket page link + passcode.
  7. Optional: set ADMIN_KEY in Apps Script and below so only you can enter match results.
